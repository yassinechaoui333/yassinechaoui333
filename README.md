## Core Competencies

**Data Engineering:**
- Pipeline orchestration with Apache Airflow, medallion architecture design (Bronze → Silver → Gold), and REST API exposure via FastAPI
- ETL development using Spark/Scala, PostgreSQL, and Docker-based deployments
- Real-time data ingestion from external APIs, data modeling, and aggregation layers

**Machine Learning:**
- Supervised learning: regression and classification pipelines following CRISP-DM methodology
- Unsupervised learning: clustering and dimensionality reduction on large tabular datasets
- Deep learning: Neural Collaborative Filtering, context fusion strategies, and ablation studies with PyTorch

**AI & Search:**
- RAG systems with pgvector embeddings and local LLMs via Ollama
- Semantic search, vector similarity, and natural language Q&A over structured data
